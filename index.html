Movies API

Documentation

I have created a database with my favorite movies!

This is how to use my API with Postman

Postman URL Description Information Method
http://localhost:3001/movies Get all Movies Shows all movies in the database, no parameters needed GET
http://localhost:3001/movie/id Get Movie by ID Shows a movie by a specific Id, Ids is an int and starting from 1 GET
http://localhost:3001/movie Add a new Movie Add a new movie to the database, requires body. Example {"title": "Harry Potter", "director": "JK Rowling"} POST
http://localhost:3001/movie/id Update a movie Update a movie in the database. Use Id to choose witch movie, and body with the updated data, example {"title": "Harry Potter", "director": "JK Rowling"} PUT
http://localhost:3001/movie/id Delete a movie Delete a movie in the database. Use Id to choose witch movie to delete DELETE
